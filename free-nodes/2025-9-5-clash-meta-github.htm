三、PandoraBox环境准备

3.1 硬件兼容性核查

建议使用以下机型：
- Newifi D2（内存消耗＜30MB）
- Xiaomi R3P（支持硬件加速）

3.2 固件升级关键步骤

  1. 通过ssh root@192.168.1.1登录后台
  2. 执行opkg update && opkg list-upgradable
  3. 使用sysupgrade -v /tmp/固件名称.bin完成升级

技术提示：升级前务必执行mtd -r backup创建完整备份

四、插件安装全流程详解

4.1 官方源安装方案

bash opkg install v2ray-core opkg install luci-app-v2ray 若出现依赖错误，需手动安装：
opkg install libopenssl libustream-openssl

4.2 离线安装应急方案

  1. 从GitHub下载v2ray_4.45.0-1_mipsel_24kc.ipk
  2. 通过WinSCP上传至/tmp目录
  3. 执行opkg install /tmp/*.ipk --force-depends

六、性能优化实战技巧

6.1 多节点负载均衡

  1. 在/etc/v2ray/config.json中添加多个outbound
  2. 配置balancer策略：
    json "balancers": [ { "tag": "balance", "selector": ["node1", "node2"] } ]

6.2 TCP优化参数

编辑/etc/sysctl.conf添加：
conf net.core.rmem_max=4194304 net.ipv4.tcp_congestion_control=bbr
